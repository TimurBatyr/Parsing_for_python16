1. Создать папку для проекта и перейти в неё
2. Создать виртуальное окружение -- "python3 -m venv <название окружение>"
3. Активировать окружение "source <название окружение>/bin/activate"

Для деактивации используется команда "deactivate"
4. Создаем файл requirements.txt и описываем используемые библиотеки
5. Скачиваем в окружение библиотеки из requirements командой "pip install -r requirements.txt"
6. Проверяем скачалось ли requirements.txt с помощью "pip freeze'
